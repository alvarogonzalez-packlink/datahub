What is DataHub?

DataHub is a modern data catalog built to enable end-to-end data discovery, data observability, and data governance. This extensible metadata platform is built for developers to tame the complexity of their rapidly evolving data ecosystems and for data practitioners to leverage the total value of data within their organization.

Key Features​

Data Discovery →Search your entire data ecosystem, including dashboards, datasets, ML models, and raw files.
Data Governance →Define ownership and track PII.
Data Quality Control →Improve data quality through metadata tests, assertions, data freshness checks, and data contracts.
UI-based Ingestion →Easily set up integrations in minutes using DataHub's intuitive UI-based ingestion feature.
APIs and SDKs →For users who prefer programmatic control, DataHub offers a comprehensive set of APIs and SDKs.
Vibrant Community →Our community provides support through office hours, workshops, and a Slack channel.
